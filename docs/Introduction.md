# Introduction

The Scientific Calculator is designed to solve mathematical problems ranging from basic arithmetic to advanced mathematics.

Unlike ordinary calculators, every mathematical topic is grouped into its own category.

Examples include:

- Trigonometry
- Logarithms
- Prime Numbers
- Complex Numbers
- Statistics
- Calculus

---

## Angle Mode

Currently the calculator uses **degrees**.

Example

sin(30)

= 0.5

---

## Mathematical Constants

The calculator already contains several predefined constants.

pi

e

phi

r15

Example

sin(pi/2)

log(e)

---

## Function Syntax

Functions are written as

function(argument)

Examples

sqrt(25)

sin(45)

gcd(24,18)

root(3,64)

---
