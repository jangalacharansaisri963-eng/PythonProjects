Scientific Calculator

A lightweight terminal-based scientific calculator written in Python.

Features

- High-precision arithmetic
- Standard arithmetic operations
- Trigonometric functions
  - "sin"
  - "cos"
  - "tan"
  - "asin"
  - "acos"
  - "atan"
- Hyperbolic functions
  - "sinh"
  - "cosh"
  - "tanh"
- Logarithmic functions
  - "ln"
  - "log"
- Roots
  - "sqrt(x)"
  - "cbrt(x)"
  - "root(n, x)"
- Factorial
  - "factorial(n)"
- Factors
  - "factors(n)"
  - "factorization(n)"
- Rational number finder
  - "fr(a, b) count"
- Mathematical constants
  - "pi"
  - "e"
- Degree mode
- Radian mode
- Optional 105-digit precise mode
- Terminal easter eggs

Installation

Clone the repository:

git clone https://github.com/jangalacharansaisri963-eng/REPOSITORY.git

Run the calculator:

python calculator.py

Or download the Windows executable from the latest successful GitHub Actions build.

Usage

Examples:

$ 2+2
4

$ sin(90)

$ sqrt(2)

$ root(5,32)

$ factorial(10)

$ factors(360)

$ factorization(360)

$ ln(e)

$ pi precise

$ fr(3,4) 10

Commands

Command| Description
"mode degree"| Enable degree mode
"mode radian"| Enable radian mode
"clear"| Clear the terminal
"exit"| Exit the calculator
"quit"| Exit the calculator
