# Prime Numbers

A prime number has exactly two positive factors.

1 and itself.

Examples

2

3

5

7

11

13

---

# is_prime(n)

Purpose

Checks whether a number is prime.

Example

is_prime(17)

=True

---

# next_prime(n)

Purpose

Returns the next prime number.

Example

next_prime(20)

=23

---

# previous_prime(n)

Purpose

Returns the previous prime number.

Example

previous_prime(20)

=19

---

# prime_factors(n)

Purpose

Returns the prime factorization.

Example

prime_factors(84)

=2×2×3×7

Remember

Keep dividing by the smallest prime.

---

# factor_count(n)

Purpose

Returns the number of factors.

Example

factor_count(12)

=6

Factors

1 2 3 4 6 12
